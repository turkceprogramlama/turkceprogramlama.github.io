Welcome to Metu-Ceng Blog Template

This is a template page for blogging using Jekyll. Creating a static website using Jekyll is very easy and Github can host Jekyll pages. You can have more information at Jekyll web site.

Before the start make sure that you have the latest version of jekyll. Run this code on terminal:

{% highlight bash %} gem install jekyll{% endhighlight %}

Please download the files at this GitHub link. This repository is a nice template to quick start and the page you are looking right now is from this template. Modify the file contents according to your need and see the website by running this:

{% highlight bash %} jekyll serve{% endhighlight %}

Congrats! It is that easy. It starts up in just 3 minutes. Go to blog to learn how to publish it on the GitHub.
